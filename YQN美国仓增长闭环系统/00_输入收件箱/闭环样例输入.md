> 以下为阶段 0 原型样例。客户名、ID、渠道 ID 均为模拟；业务场景按美国仓 / 北美跨境电商方向构造。

## IN-001 小红书素材带来的高价值美国仓 MQL

- 来源类型：小红书私信
- 线索 ID：MQL-001
- 素材 / 渠道 ID：XHS-AD-032
- 平台：TikTok Shop + Shopify
- 原始记录：客户看到小红书笔记《美国仓一件代发怎么避开旺季爆仓》后私信：“我们做手机壳和数据线，TikTok Shop 和 Shopify 都在卖，现在洛杉矶一个小仓每天 600-900 单，旺季会到 1500 单。现在最痛的是库存看不清、尾程账单乱、退货回来没法快速换标。想找能做美西美东分仓、FBM 发货和 FBA 补货的 3PL，下周可以拉运营一起聊。”
- 备注：客户主动给出品类、平台、日单量、痛点、现有履约方式和会议意向。

## IN-008 仓库 / 运营能力边界信息

- 来源类型：仓库运营内部回复
- 原始记录：仓库运营回复：“美西仓可做标准小件一件代发、FBA Prep、退货换标和基础质检；美东仓可承接备货和 B2B 转运，但拍照质检需要提前确认 SKU 和 SOP。SFP 需要单独评估截单时间和承运商组合，不能默认承诺。超大件、带电品和高价值电子配件需要先看尺寸、重量、申报信息和保险要求。”
- 备注：可用于内容边界、销售话术和报价前置字段。
